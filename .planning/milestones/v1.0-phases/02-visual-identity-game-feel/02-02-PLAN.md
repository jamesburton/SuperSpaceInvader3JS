Selective bloom approach for this codebase:
- Set renderer.layers on emissive objects to include Layer 1 (bloom layer)
- SelectiveBloomEffect targets Layer 1
- Non-emissive objects (border LineLoop, background) stay on Layer 0 only
- HUD is DOM — entirely outside WebGL pipeline, unaffected by any of this

Layer assignment: After creating entity meshes (player, enemy rowMeshes, bullet meshes),
call mesh.layers.enable(1) on each emissive mesh. Border LineLoop stays on Layer 0 only.

```typescript
import { EffectComposer, EffectPass, RenderPass, SelectiveBloomEffect } from 'postprocessing';
import type { WebGLRenderer, Scene, OrthographicCamera } from 'three';

export class BloomEffect {
  private readonly composer: EffectComposer;

  constructor(renderer: WebGLRenderer, scene: Scene, camera: OrthographicCamera) {
    this.composer = new EffectComposer(renderer);

    // Pass 1: render the scene normally
    const renderPass = new RenderPass(scene, camera);
    this.composer.addPass(renderPass);

    // Pass 2: selective bloom — only objects on layers that include Layer 1
    const bloom = new SelectiveBloomEffect(scene, camera, {
      intensity: 1.8,
      luminanceThreshold: 0.15,
      luminanceSmoothing: 0.05,
      radius: 0.75,
    });
    // SelectiveBloomEffect uses selection — add emissive objects via bloom.selection.add(mesh)
    // Exposed as public so SceneManager can pass it to entities
    this.bloomEffect = bloom;

    const effectPass = new EffectPass(camera, bloom);
    this.composer.addPass(effectPass);
  }

  public readonly bloomEffect: SelectiveBloomEffect;

  /** Call each frame instead of renderer.render() */
  public render(deltaTime: number): void {
    this.composer.render(deltaTime);
  }

  /** Call when renderer is resized */
  public setSize(width: number, height: number): void {
    this.composer.setSize(width, height);
  }

  public dispose(): void {
    this.composer.dispose();
  }
}
```

Note on SelectiveBloomEffect API: Use `bloomEffect.selection.add(mesh)` to register emissive meshes. The selection set is a THREE.Selection (Set<Object3D>). Emissive meshes must be added to this selection after creation. Non-emissive objects are not added and will not bloom.
