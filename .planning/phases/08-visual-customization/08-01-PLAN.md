All shapes must fit within the same AABB bounding box (half-extents width=20, height=12) as the existing player geometry.

Create `src/entities/PlayerSkinManager.ts` with:
- `PlayerSkinManager` class with:

The new material must use MeshStandardMaterial with color, emissive (same hex), emissiveIntensity: 1.2, roughness: 1.0, metalness: 0.0 — matching the existing Player constructor pattern.
  - Does NOT re-register with bloom (mesh reference stays the same; only geometry/material change).

Import Three.js types: Mesh, BufferGeometry, MeshStandardMaterial, Color from 'three'.
