In `src/effects/CRTManager.ts`:
- Add a private `composer: EffectComposer | null = null` field.
- In `init()`: store `this.composer = composer` before calling `composer.addPass(this.pass)`.
- In `dispose()`: call `this.composer?.removePass(this.pass!)` BEFORE calling `this.pass?.dispose()`, then null out `this.composer`. This ensures the pass is removed from the EffectComposer before being disposed, enabling clean reinit.

In `src/core/SceneManager.ts`:
- Update `initCrt()` to handle reinit (dispose old CRT before creating new):
  ```typescript
  public initCrt(tier: number | null, intensity: number): CRTManager | null {
    if (this.crtManager) {
      this.crtManager.dispose();
      this.crtManager = null;
    }
    if (!this.bloomEffect || tier === null || tier < 1) return null;
    this.crtManager = new CRTManager();
    this.crtManager.init(this.bloomEffect.composer, this.camera, tier, intensity);
    return this.crtManager;
  }
  ```
  This method is idempotent: calling it again with a different tier disposes the old CRT pass and creates a new one with the correct effects for the new tier.
