Key implementation details:

1. **init()**: Read `volume` and `muted` from `useMetaStore.getState()`. Apply to Howler globals. If the fields don't exist yet (06-01 hasn't migrated), fall back to 0.8 and false via nullish coalescing.

2. **BGM Howl**: `new Howl({ src: ['/audio/bgm-synthwave.mp3'], loop: true, volume: 1.0 })`. Do NOT pass `html5: true` — Web Audio path ensures gapless loop (AUD-01). Per-Howl volume stays at 1.0 because master volume is set globally via `Howler.volume()`.

3. **SFX Howls**: Create a Map<SfxKey, Howl> with `pool: 3` on each (rate-limiting per user decision — max 3 concurrent instances). Define SFX paths as an array of [key, basePath] tuples and iterate to create Howls.

4. **AudioContext unlock (AUD-05)**: Add a one-shot `keydown` listener in `init()` that calls `Howler.ctx?.resume()` if the AudioContext is suspended. Remove listener after first fire.

5. **playBgm()**: Calls `this.bgm?.play()`. Should be called from PlayingState.enter() (wired in 06-03).

6. **stopBgm()**: Calls `this.bgm?.stop()`. Called from GameOverState or when returning to menu.

7. **playSfx(key: SfxKey)**: Looks up the Howl in sfxMap and calls `.play()`. Silent no-op if key not found. Each `play()` call reuses or creates a sound node from the pool (capped at 3).

8. **setVolume(value: number)**: Clamps to [0, 1], calls `Howler.volume(clamped)`, calls `useMetaStore.getState().setVolume(clamped)`. Atomic update of both Howler and MetaStore (per research anti-pattern warning).

9. **setMuted(muted: boolean)**: Calls `Howler.mute(muted)`, calls `useMetaStore.getState().setMuted(muted)`.

10. **isMuted getter**: Returns `useMetaStore.getState().muted`.

11. **volume getter**: Returns current Howler volume via `Howler.volume()`.

Export the singleton: `export const audioManager = new AudioManagerClass();`
